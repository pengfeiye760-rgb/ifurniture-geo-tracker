# iFurniture GEO Performance Tracker

This is a Streamlit MVP for tracking iFurniture's GEO performance across AI search and recommendation answers.

## Current MVP Features

- AI visibility tracking
- Recommendation rank tracking
- Sentiment and risk phrase tracking
- Web-search citation source tracking
- Source domain intelligence
- Priority GEO opportunity detection
- GEO action tracker
- Action impact comparison

## Demo Scope

The current MVP uses a fixed New Zealand question bank and a pre-generated SQLite database.

The dashboard is designed to answer:

1. How often does AI mention iFurniture?
2. Where does iFurniture rank in AI recommendations?
3. Which websites does AI cite as trusted sources?
4. Which GEO actions should be prioritized?
5. How can we compare performance before and after each action?
